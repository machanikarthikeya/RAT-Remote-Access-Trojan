# Code of Conduct

## Purpose

This project is focused on ethical cybersecurity research and learning. We are committed to maintaining a respectful, responsible, and professional community.

## Expected Behavior

All participants are expected to:

- Act ethically and responsibly  
- Use this project only for legal and authorized purposes  
- Be respectful in communication  
- Provide constructive feedback  
- Follow all applicable laws and regulations  

## Unacceptable Behavior

The following actions are strictly prohibited:

- Promoting or supporting illegal activities  
- Sharing malicious use cases or harmful instructions  
- Harassment, discrimination, or abusive behavior  
- Attempting unauthorized access using this project  
- Distributing harmful or exploitative content  

## Security Responsibility

This project involves high-risk concepts. Users and contributors must:

- Use secure environments (e.g., :contentReference[oaicite:1]{index=1})  
- Avoid running on personal/main systems  
- Understand risks before usage  

## Enforcement

Violations of this Code of Conduct may result in:

- Removal of contributions  
- Temporary or permanent bans  
- Reporting to appropriate authorities if necessary  

## Disclaimer

This project is for educational and ethical purposes only. Misuse is strictly prohibited, and the maintainers are not responsible for any actions taken by users.

## Agreement

By participating in this project, you agree to follow this Code of Conduct.
